1.1- ¿Qué significa CSS?

CSS significa hojas de estilo en cascada en español (Cascading Stylesheets).

1.2- ¿Para qué sirve el CSS? ¿Qué programa se utiliza para interpretar el CSS?

CSS es un lenguaje grafico que se sirve para presentar y darle estilo al contenido que deseemos y que se vea mucho mejor esteticamente. Puede cambiar la tipografía, los colores, tamaños de imagenes o palabras, añadir animaciones, etc. Para interpretar CSS se utilizan los navegadores, como google, firefox, etc.

1.3- ¿A qué se llama regla de CSS?

Las reglas de CSS hace referencia a que cada usuario elige las reglas que serán aplicadas en el contenido, por ejemplo; "quiero que el titulo sea en letra cursiva y en color azul".

1.5- ¿Qué es un selector en CSS?

Un selector en CSS es el elemtento HTML al cual le aplicaremos el estilo deseado, por ejemplo, si quisieramos cambiar en color de algun titulo H1, nuestro selector será "h1".

1.6- Identificar y definir los selectores: de ID, de clases, universal, de etiquetas, descendente.

ID
Es un tipo de selector que será unico en el programa HTML, por ejemplo, sirve para puntualizar mas especificamente lo que queramos cambiar con CSS.
Clases
Este agrupa propiedades para poder modificarlas con CSS de manera mas facil y rapida. Ejemplo: Si tengo 20 etiqueas "h1" pero solo deseo agregarle algun estilo a 10 de ellas, la manera mas facil y rapida de hacerlo es con clases.
Universal
Este nos permite aplicar un estilo a todos los elementos HTML y se indica con un *.
Etiquetas
Este selector se le aplica a las etiquetas que queramos del proyecto HTML.

1.7- Explicar brevemente utilizando una lista ordenada: ¿cómo funciona el CSS?

  1. El navegador carga o recibe el HTML
  2. Convierte el HTML en DOM (Modelo de Objetos del Documento), este representa al documento en la memoria del ordenador.
  3. El navegador busca la mayor parte de los recursos vinculados al documento HTML.
  4. Paso "árbol de renderización", el navegador analiza el CSS, identifica los diferentes tipo de selectores, ordena en "cubos" las reglas y calcula cuales se aplicarán segun el tipo de selector.
  5. El árbol de renderización presenta la estructura en que los nodos deben aparecer después de aplicarle las reglas.
  6. Paso "pintura", se ve en pantalla, la pagina finalizada.

1.8- ¿Cuáles son las tres formas para vincular CSS con HTML?

Forma 1 - Etiqueta Style
Está dentro de "head" y se aplicará a todo nuestro HTML. Ejemplo:style p { color: red;}

Forma 2 - Atributo Style
Ésta aplicará el estilo solamente a una etiqueta en especifico, va dentro de "body".
Ejemplo: p style="color:red;">Esto es un párrafo rojo

Forma 3 - Archivo externo
En esta forma, vinculamos un archivo CSS dentro de nuestro HTML.
Ejemplo: link href="estilos.css" rel="stylesheet"
